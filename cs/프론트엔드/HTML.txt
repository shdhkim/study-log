메타데이터: 데이터를 설명해주는 데이터

반응형 웹(Responsive Web Design, RWD)은 다양한 디바이스(PC, 태블릿, 스마트폰 등)의 화면 크기에 따라
자동으로 레이아웃이 조정되는 웹 디자인 기법입니다.

SPA(Single Page Application, 단일 페이지 애플리케이션)는 한 개의 HTML 페이지에서 
동적으로 콘텐츠를 변경하면서 화면을 업데이트하는 웹 애플리케이션 방식입니다.
페이지 이동 없이도 새로운 콘텐츠를 로드할 수 있어, 빠른 사용자 경험(UX)을 제공합니다.

MPA는 요청할 때마다 새로운 HTML 페이지를 서버에서 받아오는 전통적인 웹 애플리케이션 방식입니다.

SSR은 서버에서 HTML을 완전히 렌더링한 후, 클라이언트에 전달하는 방식입니다.
클라이언트에서 HTML을 생성하는 것이 아니라 서버에서 HTML을 만들어 반환합니다.

CSR은 서버에서 기본 HTML을 제공하고, JavaScript를 통해 브라우저에서 동적으로 페이지를 렌더링하는 방식입니다.
