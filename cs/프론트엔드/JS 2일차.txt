DOM(Document Object Model)은 HTML, XML 문서를 객체(Object) 기반의 
계층적(Tree) 구조로 표현하는 프로그래밍 인터페이스(API)입니다.
웹 브라우저가 HTML 문서를 로드하면, 이를 DOM 형태로 변환하여 
JavaScript 등 프로그래밍 언어를 통해 문서를 동적으로 조작할 수 있도록 합니다.

Virtual DOM(가상 DOM)은 실제 DOM을 직접 조작하는 대신, 가상의 DOM을
메모리에서 관리하면서 변경 사항을 최소화하여 실제 DOM에 적용하는 방식입니다. 
React, Vue.js 등의 프레임워크에서 사용되며, 
DOM 업데이트 성능을 최적화하는 핵심 기술입니다.
