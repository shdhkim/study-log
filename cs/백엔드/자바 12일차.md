# Java Set, String vs StringBuilder, Stream / IntStream 정리 

## 1) Java `Set` 핵심 요약
- **중복 불가**
- `HashSet` / `LinkedHashSet` / `TreeSet`
- 중복 판단 기준: `equals()` + `hashCode()`

---

## 2) `String` vs `StringBuilder` (charAt, replace 포함)

### 2.1 String의 특징 (불변 객체)
```java
String s = "hello";
s = s.replace('h', 'H');
System.out.println(s); // Hello
```

---

### 2.2 `charAt()` 예시

```java
String s = "java";
char c1 = s.charAt(1); // 'a'

StringBuilder sb = new StringBuilder("java");
char c2 = sb.charAt(2); // 'v'
```

---

### 2.3 `replace()` 차이

```java
String s = "banana";
String r = s.replace('a', 'o');

StringBuilder sb = new StringBuilder("banana");
sb.replace(1, 3, "OO");
```

---

### 2.4 `setCharAt()` 활용
```java
StringBuilder sb = new StringBuilder("java");
sb.setCharAt(0, 'J');
System.out.println(sb); // Java
```

---

## 3) Stream vs IntStream

### 3.1 Stream<Integer>
```java
List<Integer> list = List.of(1,2,3,4,5);
int sum = list.stream()
    .filter(n -> n % 2 == 0)
    .map(n -> n * 10)
    .reduce(0, Integer::sum);
```

---

### 3.2 IntStream 기본
```java
IntStream.range(1, 6).forEach(System.out::println);
```

---

### 3.3 Stream → IntStream
```java
int sum = list.stream()
    .mapToInt(Integer::intValue)
    .sum();
```

---

### 3.4 IntSummaryStatistics
```java
IntSummaryStatistics st = IntStream.of(10,20,30)
    .summaryStatistics();
```

---

## 4) 선택 가이드
- 문자열 수정 많음 → `StringBuilder`
- 숫자 연산 → `IntStream`
