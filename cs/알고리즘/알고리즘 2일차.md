# 4. 배열 순회

## 일반 for문

```java
int[] arr = {10, 20, 30};

for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

## 코드 설명

```java
int i = 0
```

0번 인덱스부터 시작합니다.

```java
i < arr.length
```

배열 범위를 넘지 않도록 합니다.

```java
arr[i]
```

현재 인덱스의 값을 가져옵니다.

## 향상된 for문

```java
for (int x : arr) {
    System.out.println(x);
}
```

## 일반 for문과 향상된 for문 차이

| 구분        | 일반 for문       | 향상된 for문   |
| ----------- | ---------------- | -------------- |
| 인덱스 사용 | 가능             | 불가능         |
| 값만 순회   | 가능             | 가능           |
| 값 수정     | 편함             | 제한적         |
| 추천 상황   | 인덱스 필요할 때 | 값만 필요할 때 |

# 5. 배열 최대값 최소값

## 예제

```java
int[] arr = {5, 2, 9, 1, 7};

int max = Integer.MIN_VALUE;
int min = Integer.MAX_VALUE;

for (int x : arr) {
    max = Math.max(max, x);
    min = Math.min(min, x);
}

System.out.println("max = " + max);
System.out.println("min = " + min);
```

## 코드 설명

```java
int max = Integer.MIN_VALUE;
```

가장 작은 값으로 시작합니다.

그래야 어떤 값이 와도 첫 비교에서 갱신될 수 있습니다.

```java
int min = Integer.MAX_VALUE;
```

가장 큰 값으로 시작합니다.

그래야 어떤 값이 와도 첫 비교에서 갱신될 수 있습니다.

```java
max = Math.max(max, x);
```

현재 max와 x 중 더 큰 값을 max에 저장합니다.

```java
min = Math.min(min, x);
```

현재 min과 x 중 더 작은 값을 min에 저장합니다.

# 6. 배열 합계와 평균

```java
int[] arr = {10, 20, 30, 40};

int sum = 0;

for (int x : arr) {
    sum += x;
}

double avg = (double) sum / arr.length;

System.out.println(sum);
System.out.println(avg);
```

## 코드 설명

```java
sum += x;
```

현재 값을 누적합니다.

```java
(double) sum / arr.length
```

정수 나눗셈을 피하기 위해 `sum`을 double로 형변환합니다.

## 주의

```java
double avg = sum / arr.length;
```

이렇게 쓰면 정수 나눗셈이 먼저 발생할 수 있습니다.

예를 들어 `5 / 2`는 `2.5`가 아니라 `2`가 됩니다.

# 7. 배열 뒤집기

```java
int[] arr = {1, 2, 3, 4, 5};

int left = 0;
int right = arr.length - 1;

while (left < right) {
    int temp = arr[left];
    arr[left] = arr[right];
    arr[right] = temp;

    left++;
    right--;
}

System.out.println(Arrays.toString(arr));
```

## 코드 설명

```java
int left = 0;
int right = arr.length - 1;
```

왼쪽 끝과 오른쪽 끝에서 시작합니다.

```java
while (left < right)
```

두 포인터가 만나기 전까지 반복합니다.

```java
int temp = arr[left];
arr[left] = arr[right];
arr[right] = temp;
```

두 값을 교환합니다.

```java
left++;
right--;
```

왼쪽 포인터는 오른쪽으로, 오른쪽 포인터는 왼쪽으로 이동합니다.

# 10. 누적합(Prefix Sum)

누적합은 구간합을 빠르게 구하기 위한 기법입니다.

예를 들어 배열이 다음과 같다고 합시다.

```text
arr = [1, 2, 3, 4, 5]
```

0번부터 3번까지 합은:

```text
1 + 2 + 3 + 4 = 10
```

매번 반복문으로 구하면 O(N)이 걸립니다.

구간합 질문이 많으면 누적합을 만들어두는 것이 좋습니다.

---

# 11. 누적합 기본 코드

```java
int[] arr = {1, 2, 3, 4, 5};

int[] prefix = new int[arr.length + 1];

for (int i = 0; i < arr.length; i++) {
    prefix[i + 1] = prefix[i] + arr[i];
}

System.out.println(Arrays.toString(prefix));
```

출력:

```text
[0, 1, 3, 6, 10, 15]
```

## 코드 설명

```java
int[] prefix = new int[arr.length + 1];
```

누적합 배열은 보통 원본보다 1칸 크게 만듭니다.

이렇게 하면 구간합 계산이 편해집니다.

```java
prefix[i + 1] = prefix[i] + arr[i];
```

이전까지의 합에 현재 값을 더합니다.

---

# 12. 누적합으로 구간합 구하기

```java
int[] arr = {1, 2, 3, 4, 5};

int[] prefix = new int[arr.length + 1];

for (int i = 0; i < arr.length; i++) {
    prefix[i + 1] = prefix[i] + arr[i];
}

// arr[1]부터 arr[3]까지 합
int left = 1;
int right = 3;

int sum = prefix[right + 1] - prefix[left];

System.out.println(sum);
```

출력:

```text
9
```

왜 9인가?

```text
arr[1] + arr[2] + arr[3]
= 2 + 3 + 4
= 9
```

## 공식

```java
구간합(left, right) = prefix[right + 1] - prefix[left]
```

# 17. 문자 카운팅

알파벳 개수를 셀 때 배열을 사용하면 빠릅니다.

```java
String s = "banana";

int[] count = new int[26];

for (int i = 0; i < s.length(); i++) {
    char ch = s.charAt(i);
    count[ch - 'a']++;
}

System.out.println(Arrays.toString(count));
```

## 코드 설명

```java
int[] count = new int[26];
```

알파벳 소문자는 26개이므로 크기 26 배열을 만듭니다.

```java
ch - 'a'
```

문자를 숫자 인덱스로 바꿉니다.

예:

```text
'a' - 'a' = 0
'b' - 'a' = 1
'c' - 'a' = 2
```

따라서 `count[0]`은 a의 개수, `count[1]`은 b의 개수입니다.

# 20. 가장 많이 나온 알파벳 찾기

```java
public static String mostFrequentAlphabet(String str) {
    int[] count = new int[26];

    for (int i = 0; i < str.length(); i++) {
        char ch = Character.toUpperCase(str.charAt(i));
        count[ch - 'A']++;
    }

    int max = -1;
    char answer = '?';
    boolean duplicated = false;

    for (int i = 0; i < 26; i++) {
        if (count[i] > max) {
            max = count[i];
            answer = (char) (i + 'A');
            duplicated = false;
        } else if (count[i] == max) {
            duplicated = true;
        }
    }

    return duplicated ? "?" : String.valueOf(answer);
}
```

## 코드 설명

```java
int[] count = new int[26];
```

A부터 Z까지 등장 횟수를 저장합니다.

```java
char ch = Character.toUpperCase(str.charAt(i));
```

대소문자를 구분하지 않기 위해 대문자로 통일합니다.

```java
count[ch - 'A']++;
```

해당 알파벳의 카운트를 1 증가시킵니다.

```java
int max = -1;
char answer = '?';
boolean duplicated = false;
```

최대 등장 횟수, 정답 문자, 중복 여부를 저장합니다.

```java
if (count[i] > max)
```

더 많이 등장한 문자를 찾으면 정답을 갱신합니다.

```java
duplicated = false;
```

새로운 최대값이 나왔으므로 기존 중복 상태는 초기화합니다.

```java
else if (count[i] == max)
```

현재 최대 등장 횟수와 같은 알파벳이 또 있으면 중복입니다.

# 21. HashMap

`HashMap`은 key-value 구조입니다.

```java
Map<String, Integer> map = new HashMap<>();

map.put("apple", 3);
map.put("banana", 2);

System.out.println(map.get("apple"));
```

출력:

```text
3
```

## 언제 사용하나

- 문자열별 개수 세기
- 숫자별 빈도수 세기
- 이름과 점수 매핑
- 빠른 조회
- 두 값의 관계 저장

---

# 22. HashMap 주요 메서드

| 메서드                     | 의미               |
| -------------------------- | ------------------ |
| put(key, value)            | 값 저장            |
| get(key)                   | 값 조회            |
| getOrDefault(key, default) | 없으면 기본값 반환 |
| containsKey(key)           | 키 존재 여부       |
| remove(key)                | 키 삭제            |
| keySet()                   | 키 목록            |
| values()                   | 값 목록            |
| entrySet()                 | 키-값 쌍 목록      |

---

# 23. HashMap 빈도수 계산

```java
String[] arr = {"A", "B", "A", "C", "A", "B"};

Map<String, Integer> map = new HashMap<>();

for (String s : arr) {
    map.put(s, map.getOrDefault(s, 0) + 1);
}

System.out.println(map);
```

출력 예:

```text
{A=3, B=2, C=1}
```

## 코드 설명

```java
map.getOrDefault(s, 0)
```

키가 있으면 기존 값을 가져옵니다.

키가 없으면 0을 가져옵니다.

```java
map.put(s, map.getOrDefault(s, 0) + 1)
```

기존 등장 횟수에 1을 더해서 다시 저장합니다.

# 25. HashSet

`HashSet`은 중복을 허용하지 않는 자료구조입니다.

```java
Set<Integer> set = new HashSet<>();

set.add(1);
set.add(1);
set.add(2);

System.out.println(set.size());
```

출력:

```text
2
```

## 언제 사용하나

- 중복 제거
- 이미 나온 값인지 확인
- 방문 여부 체크
- 서로 다른 값의 개수 계산

---

# 26. HashSet 중복 체크

```java
int[] arr = {1, 2, 3, 2, 4};

Set<Integer> set = new HashSet<>();

for (int x : arr) {
    if (set.contains(x)) {
        System.out.println("중복 발견: " + x);
    }

    set.add(x);
}
```

## 코드 설명

```java
set.contains(x)
```

이미 x가 들어있는지 확인합니다.

```java
set.add(x)
```

현재 값을 set에 저장합니다.

중복이 있으면 `add()`를 해도 set 크기는 증가하지 않습니다.

# 29. 아나그램 배열 풀이

```java
public static boolean isAnagram(String a, String b) {
    if (a.length() != b.length()) {
        return false;
    }

    int[] count = new int[26];

    for (int i = 0; i < a.length(); i++) {
        count[a.charAt(i) - 'a']++;
        count[b.charAt(i) - 'a']--;
    }

    for (int x : count) {
        if (x != 0) {
            return false;
        }
    }

    return true;
}
```

## 코드 설명

```java
if (a.length() != b.length())
```

길이가 다르면 아나그램이 될 수 없습니다.

```java
count[a.charAt(i) - 'a']++;
```

첫 번째 문자열의 문자는 카운트를 증가시킵니다.

```java
count[b.charAt(i) - 'a']--;
```

두 번째 문자열의 문자는 카운트를 감소시킵니다.

두 문자열의 문자 구성이 같으면 최종적으로 모든 카운트가 0이 됩니다.

```java
for (int x : count) {
    if (x != 0) return false;
}
```

하나라도 0이 아니면 문자의 구성이 다르다는 뜻입니다.

# 30. 아나그램 HashMap 풀이

문자 범위가 알파벳으로 제한되지 않는다면 HashMap을 사용할 수 있습니다.

```java
public static boolean isAnagramMap(String a, String b) {
    if (a.length() != b.length()) {
        return false;
    }

    Map<Character, Integer> map = new HashMap<>();

    for (int i = 0; i < a.length(); i++) {
        char ch = a.charAt(i);
        map.put(ch, map.getOrDefault(ch, 0) + 1);
    }

    for (int i = 0; i < b.length(); i++) {
        char ch = b.charAt(i);

        if (!map.containsKey(ch)) {
            return false;
        }

        map.put(ch, map.get(ch) - 1);

        if (map.get(ch) == 0) {
            map.remove(ch);
        }
    }

    return map.isEmpty();
}
```

## 코드 설명

첫 번째 문자열의 문자 개수를 HashMap에 저장합니다.

두 번째 문자열을 돌면서 해당 문자의 개수를 하나씩 줄입니다.

모든 문자가 정확히 상쇄되면 map이 비게 됩니다.

# 31. 두 수의 합 문제

배열에서 합이 target이 되는 두 수가 있는지 확인하는 문제입니다.

## Set 풀이

```java
public static boolean twoSum(int[] arr, int target) {
    Set<Integer> set = new HashSet<>();

    for (int x : arr) {
        int need = target - x;

        if (set.contains(need)) {
            return true;
        }

        set.add(x);
    }

    return false;
}
```

## 코드 설명

```java
int need = target - x;
```

현재 값 x와 더해서 target이 되려면 필요한 값을 구합니다.

```java
set.contains(need)
```

이전에 나온 값 중 need가 있으면 정답입니다.

```java
set.add(x)
```

현재 값을 이후 비교를 위해 저장합니다.

# 32. 첫 번째 중복 문자 찾기

```java
public static char firstDuplicate(String s) {
    Set<Character> set = new HashSet<>();

    for (int i = 0; i < s.length(); i++) {
        char ch = s.charAt(i);

        if (set.contains(ch)) {
            return ch;
        }

        set.add(ch);
    }

    return '?';
}
```

## 코드 설명

문자를 왼쪽부터 보면서 이미 나온 문자인지 확인합니다.

처음으로 `contains()`가 true가 되는 문자가 첫 번째 중복 문자입니다.

# 33. 문자열 압축 기초

연속된 문자의 개수를 세는 패턴입니다.

```java
public static String compress(String s) {
    StringBuilder sb = new StringBuilder();

    int count = 1;

    for (int i = 1; i < s.length(); i++) {
        if (s.charAt(i) == s.charAt(i - 1)) {
            count++;
        } else {
            sb.append(s.charAt(i - 1)).append(count);
            count = 1;
        }
    }

    sb.append(s.charAt(s.length() - 1)).append(count);

    return sb.toString();
}
```

## 예시

```text
aaabbc
```

결과:

```text
a3b2c1
```

## 코드 설명

```java
count = 1
```

현재 연속 문자의 개수입니다.

```java
s.charAt(i) == s.charAt(i - 1)
```

현재 문자가 이전 문자와 같으면 연속된 것입니다.

```java
sb.append(s.charAt(i - 1)).append(count)
```

연속이 끊기는 순간 이전 문자와 개수를 저장합니다.

반복문이 끝난 뒤 마지막 문자 그룹은 따로 추가해야 합니다.

# 35. 자주 틀리는 부분

## 1. 배열 인덱스 범위 초과

```java
for (int i = 0; i <= arr.length; i++) {
    System.out.println(arr[i]);
}
```

틀린 코드입니다.

정답:

```java
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

## 2. substring 끝 인덱스 착각

```java
s.substring(0, 3)
```

0, 1, 2번 인덱스만 포함합니다.

3번 인덱스는 포함하지 않습니다.

## 3. 대문자인데 'a'를 빼는 실수

```java
count[ch - 'a']++;
```

ch가 대문자면 잘못된 인덱스가 나올 수 있습니다.

대문자면:

```java
count[ch - 'A']++;
```

## 4. HashMap get null 문제

```java
map.get(key) + 1
```

키가 없으면 `map.get(key)`는 null입니다.

따라서 다음처럼 써야 합니다.

```java
map.getOrDefault(key, 0) + 1
```

## 5. 배열 복사 착각

```java
int[] b = a;
```

이건 배열 복사가 아니라 같은 배열 참조입니다.

진짜 복사는:

```java
int[] b = Arrays.copyOf(a, a.length);
```
