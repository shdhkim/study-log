# 전화통신, 인터넷, 광통신, 모뎀, DTE/DCE 정리

## 1. DTE와 DCE

### DTE(Data Terminal Equipment)

데이터를 생성하거나 사용하는 장치이다.

예시 - PC - 노트북 - 서버 - 스마트폰

### DCE(Data Circuit-terminating Equipment)

DTE와 통신망을 연결해 주는 장치이다.

대표 예시 - 전화선 모뎀 - 광모뎀(ONT)

구조

```text
DTE
  │
DCE
  │
통신망
  │
DCE
  │
DTE
```

---

## 2. 모뎀(Modem)

MODEM = Modulator + Demodulator

역할은 통신 매체에 맞게 신호를 변환하는 것이다.

### 과거 전화선 인터넷

```text
PC
 │
모뎀
 │
전화선
 │
전화국
```

모뎀 역할

- 디지털 → 아날로그(변조)
- 아날로그 → 디지털(복조)

전화선은 원래 사람의 음성을 전달하도록 만들어졌기 때문에 컴퓨터 데이터는
그대로 보낼 수 없었다.

---

### 현재 광인터넷

```text
광케이블
 │
광모뎀(ONT)
 │
공유기
 │
PC
```

광모뎀 역할

- 전기신호 → 광신호
- 광신호 → 전기신호

즉, 오늘날의 광모뎀은 전화선 모뎀과 달리 광신호를 변환한다.

---

## 3. 전화통신망(PSTN)

예전 집전화 구조

```text
전화기
 │
전화선
 │
전화국
 │
상대 전화기
```

특징

- 회선교환(Circuit Switching)
- 인터넷 사용 안 함
- 통화 동안 회선을 계속 점유

---

## 4. 예전 인터넷

초기 인터넷은 전화선을 이용했다.

```text
PC
 │
모뎀
 │
전화선
 │
ISP
 │
인터넷
```

인터넷을 사용하는 동안 전화선을 점유하여 동시에 전화 사용이 어려운
경우가 많았다.

---

## 5. ADSL

하나의 전화선에서

```text
전화
인터넷
```

을 동시에 사용할 수 있도록 발전하였다.

---

## 6. 현재 인터넷

현재 대부분 FTTH(Fiber To The Home)를 사용한다.

```text
인터넷
 │
광케이블
 │
광모뎀(ONT)
 │
공유기
 ├─ PC
 ├─ TV
 └─ 스마트폰
```

---

## 7. 광통신

광통신은 빛을 이용하여 데이터를 전달하는 기술이다.

전송 과정

```text
컴퓨터
 │
전기신호
 │
레이저
 │
빛
 │
광섬유
 │
빛
 │
포토다이오드
 │
전기신호
 │
컴퓨터
```

광섬유는

- 코어
- 클래딩
- 보호층

으로 구성된다.

빛은 전반사(Total Internal Reflection)를 이용하여 광섬유 내부를
이동한다.

장점

- 매우 빠른 속도
- 장거리 전송
- 전자기 간섭에 강함
- 대역폭이 매우 큼

---

## 8. 휴대폰 통신

휴대폰은 전화선을 연결할 수 없으므로 전파를 사용한다.

```text
휴대폰
 │
전파
 │
기지국
 │
광케이블
 │
통신사
```

기지국까지만 무선이며 그 이후는 대부분 광케이블을 이용한다.

---

## 9. VoLTE

일반 스마트폰 통화는 대부분 VoLTE 또는 VoNR을 사용한다.

구조

```text
휴대폰
 │
LTE/5G
 │
기지국
 │
통신사 IP망(IMS)
 │
상대방
```

특징

- IP 기반
- 공용 인터넷이 아니라 통신사의 전용 IP망 사용
- 품질(QoS) 보장
- 긴급전화 지원

---

## 10. 카카오톡 통화

```text
휴대폰
 │
LTE/5G 또는 Wi-Fi
 │
공용 인터넷
 │
카카오 서버
 │
공용 인터넷
 │
상대방
```

VoLTE와 달리 공용 인터넷을 사용한다.

---

## 11. IP와 인터넷의 차이

IP는 데이터를 전달하는 규칙(프로토콜)이다.

인터넷은 IP를 사용하는 거대한 공용 네트워크이다.

예를 들어

- 회사 내부망도 IP 사용
- 통신사 내부망도 IP 사용
- 인터넷도 IP 사용

즉, IP를 사용한다고 모두 인터넷은 아니다.

## 핵심 요약

1.  DTE는 데이터를 생성하는 장치이고 DCE는 통신망과 연결하는 장치이다.
2.  예전 모뎀은 디지털과 아날로그를 변환했다.
3.  현재 광모뎀은 전기신호와 광신호를 변환한다.
4.  예전 인터넷은 전화선을 사용했다.
5.  현재 인터넷은 대부분 광케이블을 사용한다.
6.  휴대폰은 기지국까지만 전파를 사용하고 이후는 대부분 광케이블이다.
7.  VoLTE는 IP를 사용하지만 공용 인터넷이 아닌 통신사의 전용 IP망을
    사용한다.
8.  카카오톡 통화는 공용 인터넷을 이용하는 VoIP이다.
9.  IP는 프로토콜이고 인터넷은 공용 네트워크이다.
