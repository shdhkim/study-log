# 통신 기초 정리

---

# 1. 사인파(Sine Wave)

## 정의

가장 기본적인 파형으로, 대부분의 통신에서 반송파로 사용된다.

## 구성 요소

- **진폭(Amplitude)** : 파형의 높이
- **주파수(Frequency)** : 1초 동안 반복되는 횟수(Hz)
- **위상(Phase)** : 파형이 시작하는 위치

예시

```
      /\      /\
     /  \    /  \
____/    \__/    \____
```

---

# 2. 반송파(Carrier)

## 정의

정보를 실어 나르는 **고주파 사인파**이다.

반송파 자체에는 정보가 없다.

```
~~~~~~~~~~~~~~
```

여기에 음성이나 데이터를 실어서 전송한다.

예시

```
음성

↓

반송파(100MHz)

↓

변조

↓

송신
```

---

# 3. 변조(Modulation)

## 정의

정보를 반송파에 실는 과정

## 변조를 하는 이유

### ① 장거리 전송

저주파 신호를 그대로 보내기 어렵다.

고주파 반송파에 실어서 보낸다.

---

### ② 안테나 크기 감소

주파수가 높을수록 안테나를 작게 만들 수 있다.

---

### ③ 여러 사용자 동시 통신

예시

```
A 방송 → 91.9MHz

B 방송 → 95.1MHz

C 방송 → 107.7MHz
```

라디오는 원하는 주파수만 선택한다.

---

### ④ 잡음에 강한 통신

- AM
- FM
- QAM

등 다양한 변조 방식을 사용할 수 있다.

---

### ⑤ 디지털 데이터 전송

Wi-Fi

LTE

5G

모두 변조를 이용한다.

---

# 4. 증폭(Amplification)

## 정의

신호의 **진폭만** 크게 만드는 과정

증폭 후

- 진폭 : 증가
- 주파수 : 그대로
- 위상 : 그대로

예)

```
원래

   /\
__/  \__

↓

증폭

      /\
_____/  \_____
```

# 6. 푸리에 변환(Fourier Transform)

## 정의

복잡한 신호를

여러 개의 주파수로 분해하는 기술

예)

시간 영역

```
/\/\_/\/\___
```

↓

주파수 영역

```
100Hz

500Hz

1000Hz
```

활용

- 통신
- 음성
- 영상
- Wi-Fi
- LTE
- 5G

# 7. HDLC

## 정의

OSI 2계층(Data Link Layer)의

비트 지향(Bit-Oriented) 프로토콜

## 역할

- 프레임 생성
- 오류 검출
- 흐름 제어
- 재전송
- 시작/종료 구분

---

## 프레임 구조

```
Flag

Address

Control

Data

FCS

Flag
```

---

## Flag

항상

```
01111110
```

이다.

역할

```
Flag

↓

프레임 시작
```

```
Flag

↓

프레임 종료
```

---

## Address

누구에게 보내는 프레임인지 나타낸다.

예)

```
PC1

PC2

PC3
```

Address = PC2

이면

PC2만 데이터를 처리하고

나머지는 버린다.

---

## Control

제어 정보

예)

- ACK
- 순서번호
- 재전송

---

## Data

실제 데이터

예)

```
HELLO
```

---

## FCS

CRC 오류검출

송신

```
데이터

↓

CRC 계산

↓

전송
```

수신

```
CRC 계산

↓

비교

↓

같음 → 정상

다름 → 오류
```

---

## Bit Stuffing

Flag와 같은 패턴이 데이터 안에 생기지 않도록

1이 5개 연속되면

0을 하나 삽입한다.

예)

원래

```
111111
```

전송

```
1111101
```

수신

```
111111
```

복원

# 10. Wireshark

## 정의

네트워크 패킷 분석 프로그램(Packet Analyzer)

## 역할

패킷을 캡처하여

각 계층 정보를 보여준다.

## 확인 가능한 정보

- Ethernet Frame
- Wi-Fi Frame
- MAC Address
- IP Address
- TCP
- UDP
- Port
- DNS
- HTTP
- HTTPS
- TLS Handshake
- TCP 3-Way Handshake
- SYN
- ACK
- FIN
- 재전송(Retransmission)
- 패킷 크기
- 응답 시간

## 사용 목적

- 네트워크 장애 분석
- 패킷 확인
- TCP 통신 확인
- HTTP 요청/응답 확인
- DNS 조회 확인
- 성능 분석

# 핵심 요약

- **사인파** : 가장 기본적인 파형
- **반송파** : 정보를 실어 나르는 고주파 사인파
- **변조** : 정보를 반송파에 실는 과정
- **증폭** : 진폭만 크게 만드는 과정
- **복조** : 원래 정보를 복원하는 과정
- **푸리에 변환** : 신호를 주파수 성분으로 분해
- **HDLC** : 데이터링크 계층 프레임 프로토콜
- **Ethernet** : 유선 LAN
- **Wi-Fi** : 무선 LAN
- **Wireshark** : 네트워크 패킷 분석 도구
