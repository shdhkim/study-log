# PCM / DPCM / ADPCM / DM / ADM 완전 정리

# 왜 이런 기술이 필요한가?

사람의 음성은 아날로그 신호이다.

예를 들어 사람이 "안녕하세요"라고 말하면 공기의 압력이 연속적으로
변한다.

즉,

- 시간도 연속적
- 전압도 연속적

인 신호이다.

하지만 컴퓨터는

- 0
- 1

만 저장할 수 있다.

따라서 음성을 디지털 데이터로 변환해야 한다.

그 과정이

1.  표본화(Sampling)
2.  양자화(Quantization)
3.  부호화(Encoding)

이다.

---

# 표본화(Sampling)

연속적인 음성 신호를 일정 시간 간격으로 측정하는 과정

예)

원래 음성

시간에 따라 계속 변화

측정 결과

t1 : 3.00V t2 : 3.27V t3 : 3.60V t4 : 3.55V

이렇게 일정 시간마다 값을 기록한다.

---

# 왜 최고 주파수의 2배 이상으로 샘플링할까?

나이퀴스트 정리

fs ≥ 2fmax

- fs : 샘플링 주파수
- fmax : 최고 주파수

예)

최고 주파수 4kHz

샘플링 주파수 최소 8kHz

의미

1초에 4000번 진동하는 신호라면

최소한

1초에 8000번

측정해야 원래 신호를 복원 가능하다.

---

# 전화 음성이 8kHz인 이유

사람 음성은 보통

300Hz \~ 3400Hz

정도만 사용한다.

최고 주파수

3400Hz

나이퀴스트 적용

6800Hz 이상 필요

실제로는

8000Hz

사용

즉

1초에 8000번 측정

---

# 양자화(Quantization)

측정한 값을 가장 가까운 디지털 단계로 변환

예)

측정값

3.27V

양자화 단계

3.0V 3.5V 4.0V

결과

3.27V → 3.5V

---

# 양자화 비트 수

한 샘플을 몇 비트로 표현할지 결정

단계 수 = 2\^n

예)

1비트 → 2단계 2비트 → 4단계 3비트 → 8단계 8비트 → 256단계 16비트 →
65536단계

비트 수가 높을수록

- 오차 감소
- 음질 향상

---

# PCM (Pulse Code Modulation)

실제 샘플값을 양자화하여 전송

원본

100 102 103 105

PCM

100 102 103 105

특징

- 음질 좋음
- 데이터 많음

---

# DPCM (Differential PCM)

현재 값이 아니라

이전 값과의 차이값 전송

원본

100 102 103 105

DPCM

100 +2 +1 +2

특징

- 데이터 감소
- 음성처럼 연속적으로 변하는 신호에 유리

---

# 왜 차이값이 작을까?

음성은 대부분 연속적으로 변한다.

예)

100 101 102 101 103

처럼 조금씩 변화

갑자기

100 5000 -3000

처럼 바뀌지 않는다.

따라서

실제값보다

차이값이 훨씬 작다.

그래서 적은 비트로 표현 가능

---

# ADPCM (Adaptive DPCM)

DPCM 개선판

차이값을 적응적으로 양자화

예)

변화가 작음

+1 +2 +1

→ 작은 간격 사용

변화가 큼

+100 +120

→ 큰 간격 사용

특징

- PCM보다 압축률 높음
- 음질 유지

---

# DM (Delta Modulation)

차이값 크기도 안 보냄

증가/감소만 보냄

예)

100 101 102 101

DM

↑ ↑ ↓

또는

1 1 0

---

# ADM (Adaptive Delta Modulation)

DM 개선판

신호 변화가 빠르면

큰 스텝 사용

신호 변화가 느리면

작은 스텝 사용

특징

- DM보다 정확
- 데이터 매우 적음

---

# 데이터율 공식

R = fs × n

- R : 전송속도(bps)
- fs : 샘플링 주파수(Hz)
- n : 양자화 비트 수(bit)

예)

8kHz 8bit

R

8000 × 8

= 64000bps

= 64kbps

---

# 단위 정리

## Hz

주파수 단위

1Hz

1초에 1번 반복

예)

100Hz

1초에 100번 진동

1000Hz

= 1kHz

1000kHz

= 1MHz

1000MHz

= 1GHz

---

## bps

전송속도 단위

1bps

1초에 1비트

1000bps

= 1Kbps

1000Kbps

= 1Mbps

1000Mbps

= 1Gbps

---

# 핵심 암기

PCM = 실제값 전송

DPCM = 차이값 전송

ADPCM = 적응형 차이값 전송

DM = 증가/감소 전송

ADM = 적응형 증가/감소 전송

음성은 연속적으로 변하기 때문에

PCM보다

DPCM, ADPCM이 압축 효율이 좋다.

샘플링 주파수

≥ 최고 주파수 × 2

양자화 단계 수

= 2\^n

데이터율

= 샘플링 주파수 × 양자화 비트 수
