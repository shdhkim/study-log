# 데이터 결합 방식

## Merge
- **열(Column)을 기준으로 데이터를 결합**합니다.

## Join
- **인덱스(Index)를 기준으로 데이터를 결합**합니다.

# 데이터 스케일링

## 정규화 (MinMax Scaling)
- **최소값**은 0, **최대값**은 1로 변환
- 데이터 간 **상대적 비율을 유지**
- 범위가 \[0, 1\] 사이로 제한되므로 **신경망**, **KNN** 등에 적합

## 표준화 (Standard Scaling)
- **평균(Mean)이 0**, **표준편차(Standard Deviation)가 1**이 되도록 변환
- 이상치(Outlier)에 **덜 민감**
- **선형 회귀, PCA** 등에 적합
