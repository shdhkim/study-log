instance 개별 실행 단위를 표현하는 개념적 표현

service 특정기능을 수행하는 것을 의미하며 개념적 표현 

application 사용자의 특정 목적을 수행하는 소프트웨어 프로그램 

node 노드는 명시적으로 분리되어 있는 하나의 단위 표현 

cluster 클러스터는 여러 개의 노드를 하나의 시스템처럼 운영하는 구성 방식 

instance와 node는 비슷한데 조금 다름 

네트워크 노드: 물리 장비 한 대
AWS Node 그룹: 컴퓨터 여러 대 

gateway는 관문 

router는 연결 

load balancer 요청 트래픽을 여러 노드에 분배/분산 
gateway>=load balancer>router
